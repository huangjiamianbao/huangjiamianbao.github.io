喜+1导致找不到自己想玩的游戏怎么办？试试Depressurizer的自动分类吧。

我刚入坑的时候还不知道慈善包和喜+1这种操作，库里只有几十个游戏，因此不用进行分类管理也一目了然。自从开始喜+1，游戏数量就开始飙升，导致我很多时候找不到自己想玩的游戏。终于，我下定决心对游戏进行分类。
一开始，我将游戏分为了喜+1，独立游戏，大作等分类，但带来的问题就是有的游戏并不知道是否是喜+1得来的，况且这样分类每类里仍然有很多游戏，标签模糊导致效率并没有很大提升。直到我发现了一款Steam游戏自动分类的软件——Depressurizer。

Depressurizer is a program aimed at making it a bit easier to manage large Steam game libraries.

In addition to providing a way to quickly and easily modify games’ assigned categories, it also lets you mark them as Favorites or as hidden.

Depressurizer can also autocategorize your games for you. Currently, it does so based on data from that games’ Steam store pages. It can use genres, store categories (like “Single-Player” and “Steam Cloud”) or tags.

It also saves your configuration information independently of Steam, providing an automatic backup in the event that Steam loses your configuration.

下载

在GitHub下载，目前最新版是0.6.1.0，下载第一个zip即可。

Requirements

Depressurizer requires the .NET Framework 4.5.1.

It must be run on the same computer on which you use Steam, or one one of the computers on which you use Steam.

解压后得到Depressurizer.exe一路打开。途中Depressurizer会自动检测你的Steam地址和Steam账户信息，如果信息有误，手动更改即可。

更新游戏信息

成功打开后，要想自动分类，首先要检测游戏信息。菜单栏选择Tools，再选择Datebase Editor。然后如图所示更新游戏信息。

设置自动分类

第二步，设置自动分类，告诉Depressurizer你的分类标准。菜单栏选择Profile，再选择Edit Autocats。
由于个人分类方式不同，请自定义你的分类标准。我的分类标准：游戏分类，发行年份，用户评测。仅供参考。

自动分类

制订好分类标准后，让Depressurizer开始分类。菜单栏选择Tools，再选择Auto-Cat ALL中你想使用的分类标准（可多选）。

保存

保存前退出你的Steam！

确认分类无误后，菜单栏选择File，再选择Save Profile。退出Depressurizer，即可打开Steam欣赏你的游戏分类。

其他

  • Depressurizer的分类保存在本地，因此，你可能需要在你的所有电脑上运行。
  • 每次喜+1后，请重复以上步骤。
  • 无法分类的游戏（如锁区，信息不详）将会被归类到游戏分类。
  • Depressurizer的bug较多，如有错误请多次尝试。
