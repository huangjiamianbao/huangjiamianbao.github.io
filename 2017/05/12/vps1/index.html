理论基础

VPS？

可以简单地理解为小型服务器？

墙？

翻墙？

绕过防火墙，使用谷歌，维基百科，pornhub等良心产品

shadowsocks

一种翻墙方法

为什么不用其他方法？（ss的优点）

  • 安全（相对于open VPN）
  • 简单（相对于黑科技）
  • 网络质量好（几乎满速）
  • 完整的控制权（相对于第三方VPN）
  • 全平台（Windows，macOS，iOS，android等）

缺点

  • DIY
  • VPS的钱
  • ss客户端的钱（iOS）

服务端

你需要一台海外的vps
这里我推荐的（现在用的）是vultr的日本东京机房，当然你要说我做广告我也没办法(╯°□°）╯︵ ┻━┻
vultr
如果用这个链接注册我似乎能拿到钱？
注册完后会让你先充一笔钱，最低$5（支持支付宝）
充完钱后账户激活，就可以开始部署VPS了
点击加号进入VPS规格挑选

  1. 类型选择Vultr Cloud Compute(VC2)
  2. 地点建议选择日本东京（低ping）
  3. 系统选择64 bit OS中的Centos 6 x64
  4. 配置选择最低配就行

    $5/月（按小时计费）
    25G SSD
    1 CPU
    1G 内存
    1000G 流量

  5. 下面的选项不用更改，点击Deploy Now
    稍等一会，VPS就会安装好系统，并且你会收到一封邮件
    在vultr中找到你的VPS，你需要记下三个信息
  • IP
  • 用户名（都是root）
  • 密码
