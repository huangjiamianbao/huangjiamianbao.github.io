本篇文章面向的是

  1. steamer 且
  2. 了解steam集换式卡牌 且

  1. 如果你不是steamer

  3. 本地ASF挂卡

    （入门）ASF图形配置程序设置

附上我自己的ASF配置文件，仅供参考 huangjiamianbao.json

安装mono（以centos6为例）

ASF在Windows上运行需要.NET，而在centos下则需要mono

yum install mono-complete

检查mono -V

参见Mono

上传ASF

上传ASF.exe和config文件夹到VPS

Windows

FlashFXP，官网疑似gg，自行下载
在站点管理器中填写好信息后上传

android

如果已在墙外，Google Play下载AndFTP
否则，AndFTP

填写好域名，类型，用户名，密码后上传

其他系统

未知

mono运行ASF

screen -S asf
后台运行ASF

cd ASF
打开ASF目录

mono ASF.exe
使用mono运行ASF

ASF成功打开后（可能需要输入手机令牌），就可以关闭连接

后期维护

screen -ls
查看后台

kill 进程号
关闭后台ASF

参照上一部分重启ASF
