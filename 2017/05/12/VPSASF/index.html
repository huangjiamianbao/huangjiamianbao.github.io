安装mono（以centos6为例）

ASF在Windows上运行需要.NET，而在centos下则需要mono

yum install mono-complete

检查mono -V

参见Mono

上传ASF

上传ASF.exe和config文件夹到VPS

Windows

FlashFXP，官网疑似gg，自行下载
在站点管理器中填写好信息后上传

android

Google Play下载AndFTP
或在此下载AndFTP

填写好域名，类型，用户名，密码后上传

其他系统

未知

mono运行ASF

screen -S asf
后台运行ASF

cd ASF
打开ASF目录

mono ASF.exe
使用mono运行ASF

ASF成功打开后（可能需要输入手机令牌），就可以关闭连接

后期维护

screen -ls
查看后台

kill 进程号
关闭后台进程

screen -r 进程名
恢复后台进程
